Générer le manifest d'images

Le bouton Rafraîchir recharge le fichier img/manifest.json en mémoire dans la page, mais il ne peut pas créer ou modifier ce fichier depuis le navigateur.

Pour inclure une nouvelle image ajoutée dans le dossier img/, exécute cette commande dans le dossier du projet :

npm run manifest

Ou, si tu n'utilises pas npm :

node generate-manifest.js

Après avoir exécuté cette commande, clique à nouveau sur le bouton Rafraîchir dans le formulaire pour voir les nouvelles images.

Le script génère automatiquement un manifeste trié avec tous les fichiers images/vidéo valides dans img/, sauf manifest.json.
